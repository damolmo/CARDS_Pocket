# UNO Pocket
Juego de cartas multiplataforma escrito en Flutter

# Compatibilidad
- [x] android
- [x] iOS
- [x] macOS
- [x] web

# Dependencias
Las dependencias de software libre que hacen que este proyecto sea posible
```
  stacked: ^3.2.0
  audioplayers: ^3.0.1
  flutter_launcher_icons: ^0.12.0
  sqflite: ^2.2.6
  quiver: ^3.2.1
  path_provider: ^2.0.14
```
